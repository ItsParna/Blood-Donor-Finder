# Blood-Donor-Finder
This is a idea of solving real life problem in medical emergency.
Blood Donor Finder is a php based web project with user friendly layouts.
# Description
This is a prototype of a website which helps you to find your nearby blood donor in case of madical emergency.You can simply open the website and register your details like 
1) Your name - For user identity
2) Your Email - For sending details & essential information
3) Your Blood Group - For knowing the needed blood group 
4) Your location - For reaching your nearest blood donor
5) Your phone number - To contact with the donor
6) Password - For security purpose

After registering you can simply search for the nearest donor and the website provides you the secured & right details of the donor with their phone numbe , address , name .
